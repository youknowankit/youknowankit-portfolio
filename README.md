# Portfolio Website — Ankit Yadav

## 🚀 Project Overview

This is my personal portfolio website built using **React**, **Tailwind CSS**, and **Lucide-React**. It showcases my journey, skills, projects, and contact details. The design is fully responsive, visually clean, and optimized for both mobile and desktop screens.

## 🧰 Tech Stack

- **Frontend:** React, JavaScript (ES6+), JSX
- **Styling:** Tailwind CSS, CSS
- **UI Libraries / Assets:** Lucide-React, local image assets for logos & pics
- **Features:** Responsive layout, smooth navigation, project cards, contact form with mailto link

## 📄 Sections

- **Hero** — Intro with name, role, short bio & CTA to download CV
- **About** — Brief about me, skills & journey description
- **Projects** — Grid of project cards with live preview & GitHub links
- **Contact** — Contact form (name, email, message) that opens mailto link
- **Footer** — Basic footer with copyright
